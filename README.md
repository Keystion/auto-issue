# generate-issue-based-on-sitemap

## config file

```js
const config = {
  username: '', // github账号，对应Gitalk配置中的owner
  repoName: '', // 用于存储Issue的仓库名，对应Gitalk配置中的repo
  token: '', // 前面申请的personal access token
  sitemapUrl: '', // 自己站点的sitemap地址
};

module.exports = config;
```

## Get started

```
npm run dev
```
